# API Documentation

## Authentication APIs

### Register User
POST /api/auth/register

### Login User
POST /api/auth/login

---

## Employee APIs

### Get Employees
GET /api/employees

### Add Employee
POST /api/employees

### Update Employee
PUT /api/employees/{id}

### Delete Employee
DELETE /api/employees/{id}

---

## Attendance APIs

### Mark Attendance
POST /api/attendance

### Get Attendance
GET /api/attendance

## Leave APIs
