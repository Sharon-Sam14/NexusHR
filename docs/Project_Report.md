# NexusHR Project Report

## Introduction

NexusHR is a full-stack HR management platform developed using Java Spring Boot and React.

The system helps organizations manage:
- Employees
- Attendance
- Payroll
- Leave Requests
- Workforce Analytics

---

## Objectives

- Automate HR processes
- Reduce manual record handling
- Improve employee management
- Provide analytics dashboard
- Secure employee data using JWT authentication

---

## Modules

- Authentication Module
- Employee Management Module
- Attendance Module
- Leave Management Module
- Payroll Module
- Dashboard & Analytics Module

---

## Technologies Used

### Backend
- Java 21
- Spring Boot
- Spring Security
- PostgreSQL

### Frontend
- React
- Vite
- Tailwind CSS

---

## Conclusion

NexusHR provides a scalable and modern HR management solution with secure authentication and enterprise-inspired architecture.
