# System Architecture

## Frontend
React + Vite application used for:
- User Interface
- Dashboard
- Employee Management
- Attendance
- Payroll

---

## Backend
Spring Boot REST API used for:
- Business Logic
- Authentication
- Database Operations
- Security

---

## Database
PostgreSQL database used for storing:
- Users
- Employees
- Attendance
- Payroll
- Leave Requests

---

## Architecture Flow

Frontend → REST API → Service Layer → Repository Layer → PostgreSQL Database
